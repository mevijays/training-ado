# GenAI for DevOps Engineers

A short, opinionated track on **generative AI** and **AI coding agents** for DevOps engineers.

## Module index

| # | Module | What it covers |
|---|---|---|
| 1 | [GenAI Fundamentals](01-genai-fundamentals.md) | What generative AI is, key terms |
| 2 | [LLMs Explained](02-llms-explained.md) | How a large language model actually works |
| 3 | [Coding Agents Overview](03-coding-agents-overview.md) | GitHub Copilot, Claude Code, Cursor, Antigravity |
| 4 | [GenAI in DevOps Workflows](04-genai-in-devops.md) | Pipeline drafting, PR review, KQL, incident response |

## How to use this section

- Students: read in order, ~10-15 min each.
- Trainers: weave examples from modules 3 and 4 into the relevant course modules (e.g., show Copilot writing Bicep when teaching module 15).
